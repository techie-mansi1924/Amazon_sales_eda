# 📊 Amazon Product Data Analysis (EDA)

## 📌 Overview
This project performs Exploratory Data Analysis (EDA) on an e-commerce dataset of Amazon products.  
The goal is to understand pricing trends, discount strategies, and customer behavior.

---

## 📁 Dataset Description
The dataset includes:
- Product name and category  
- Actual price and discounted price  
- Discount percentage  
- Ratings and rating count  
- User reviews  

---

## 🛠 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## 🔧 Data Cleaning
- Removed currency symbols (₹) and commas  
- Converted price and rating columns to numeric  
- Handled missing values in rating and rating_count  

---

## 📊 Key Insights
- Most products are in the low to mid price range  
- Discounts have little impact on ratings  
- Ratings are generally high (positive bias)  
- Few products dominate customer reviews  
- No strong relationship between price and rating  

---

## 📈 Project Workflow
1. Data Cleaning  
2. Feature Engineering  
3. Data Visualization  
4. Insight Generation  

---

## 🚀 Conclusion
This project highlights real-world e-commerce trends, showing that discounts do not strongly influence customer satisfaction, while product popularity is concentrated among a few items.
