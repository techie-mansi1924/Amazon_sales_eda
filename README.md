📊 Amazon Product Data Analysis (EDA)
📌 Overview
This project performs Exploratory Data Analysis (EDA) on an e-commerce dataset of Amazon products. The goal is to understand pricing trends, discount strategies, and customer behavior using Python.

📁 Dataset Description
  The dataset contains information about:
  Product name and category
  Actual price and discounted price
  Discount percentage
  Product ratings and rating count
  User reviews

🛠 Tools & Technologies
  Python
  Pandas
  NumPy
  Matplotlib
  Seaborn
  Jupyter Notebook

 Data Cleaning
   Removed currency symbols (₹) and commas from price columns
   Converted prices and ratings into numeric format
   Handled missing values in rating and rating_count



📊 Key Insights
  Most products fall in the low to mid price range
  Discounts are common but have little impact on ratings
  Ratings are generally high (positive bias)
  A small number of products dominate customer reviews
  No strong relationship between price and rating



📈 Project Workflow
  Data Loading
  Data Cleaning & Preprocessing


Feature Engineering (discount calculation)
  Data Visualization
  Insight Generation
 Conclusion
   The analysis highlights real-world e-commerce trends, showing that pricing strategies and discounts do not always influence customer satisfaction, while product popularity is concentrated among a few items.
